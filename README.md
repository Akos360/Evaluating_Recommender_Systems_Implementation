## Setup

1. **Create a virtual environment** (recommended)

**For macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```
**For Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

2. **Install requirements**
```bash
pip install -r requirements.txt
```
